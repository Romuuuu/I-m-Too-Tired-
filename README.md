# SO TIRED! - Stardew Valley Mod

**SO TIRED!** is a Stardew Valley mod that makes the game stamina management more challenging, immersive, and dynamic. It punishes players for overworking and staying up late, while rewarding them for resting early!

## 🌟 Features

### 1. Dynamic Stamina Cost
Swinging tools (Pickaxe, Axe, Hoe, Weapons, Fishing Rod, etc.) now costs additional stamina! 
* The extra stamina cost scales depending on your configuration.
* **Skill Progression:** As you level up your skills (e.g., Mining level for the Pickaxe), you gain proficiency which significantly reduces the stamina cost of your tools.
* **Random Zero Cost:** Sometimes, you'll get lucky and a tool swing won't cost any extra stamina at all!

### 2. Sleep Penalties
Going to bed late has real consequences:
* If you go to bed after **11:00 PM**, you will wake up with less stamina the next day.
* If you go to bed after **12:00 AM**, the penalty is even steeper.
* **Sore Body:** If a penalty is applied, you will receive a morning popup letting you know your body is sore and you didn't sleep well. Your tool stamina costs will also be multiplied for the entire day!

### 3. Early Bedtime Bonus (Great Resting)
Hard work deserves a good rest! 
* If you go to bed early (before **6:00 PM**) and you aren't completely exhausted, you will wake up the next morning fully recovered with 100% stamina.
* You will also be rewarded with the **Great Resting** buff, giving you a boost for the first few minutes of your day!

### 4. Grace Period
Pushing yourself to the absolute limit? 
* If your stamina drops into the critical threshold (default 10%) while working, you will activate a **Grace Period**.
* This buff grants you temporary invincibility, giving you a brief window to eat food or escape danger before you pass out or take damage.

## 🛠️ Installation
1. Install the latest version of [SMAPI](https://smapi.io/).
2. Download the `SoTired 1.0.0.zip` release file.
3. Extract the `.zip` file into your `Stardew Valley/Mods` folder.
4. Launch the game using SMAPI!

## ⚙️ Configuration
Upon running the mod for the first time, a `config.json` file will be generated inside the mod folder. You can tweak almost every aspect of the mod, including multipliers, buffs, and time thresholds!

This mod support stardewvalley 1.16+
